# Dynamic Traffic RL Logistics Simulator

This project upgrades a basic RL logistics path planning system into a dynamic traffic simulation environment.

## Features

- Graph-based logistics routing environment
- Dynamic traffic cost changes for each episode
- Q-Learning agent for adaptive routing
- Comparison with Dijkstra shortest path
- Training reward visualization
- Path cost comparison across multiple runs
- Automatic saving of output images
